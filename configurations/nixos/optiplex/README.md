# Optiplex (Dell Optiplex 5060)

- Mini PC intended for offsite backup solution
- Small amount of storage planned (6TB) with no redundancy

## Services

This server runs various network services and monitoring tools:
- **Dashy**: Service dashboard
- **Glances**: System monitoring
- **Uptime Kuma**: Uptime monitoring
- **Authelia**: SSO authentication and authorization

For detailed Authelia SSO setup, see [services/AUTHELIA_SETUP.md](./services/AUTHELIA_SETUP.md).
